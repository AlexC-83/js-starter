# js-starter

## Setup

### Installing Node

Install Node >=18

https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating

Clone this repo

```
git clone https://github.com/AlexC-83/js-starter.git
```

Install node modules

```
npm install
```

Run the example unit test

```
npm test
```
